# Whitney Biennial Galleries VR Tour

This project is a browser-based virtual tour of selected galleries from the Whitney Biennial (2026), designed to make contemporary art more accessible to neurodivergent visitors. Inspired by the official exhibition [Whitney Biennial 2026](https://whitney.org/exhibitions/2026-biennial), the experience recreates parts of the museum through 360° photography and interactive hotspots, allowing users to explore artworks in a self-paced, low-pressure environment.

Traditional museum spaces can be overwhelming due to crowds, noise, and sensory intensity. This project responds to that challenge by offering an alternative mode of engagement. We prioritize comfort and autonomy. Users can access the experience anytime, from anywhere, before, during, or after a physical visit.

The interface emphasizes simplicity and accessibility. Content is written in plain language, available in both English and Spanish, and includes adjustable visual settings such as color muting. These features aim to reduce cognitive load while maintaining meaningful interaction with the artwork.

This project extends aims in creating a more inclusive entry point for audiences who are often underserved in traditional exhibition design.
